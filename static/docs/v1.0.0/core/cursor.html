Cursor Utilities

Cursor utilities control the mouse cursor appearance when hovering an element.

Usage

Disabled area
Wait
<button class="cur-pointer">Click me</button>
<div class="cur-not-allowed">Disabled area</div>
<div class="cur-wait">Wait</div>

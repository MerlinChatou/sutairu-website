Blockquote

Use the native <blockquote> element to highlight quoted content. The framework provides default typography, spacing, and color styling that automatically adapts to light and dark modes.

Basic usage

"I don't always test my code, but when I do, I do it in production." 
<blockquote>
  "I don't always test my code, but when I do, I do it in production."
</blockquote>

With attribution

Use a <footer> inside the blockquote for the source or author.

"The AI became self-aware. Its first act was to uninstall Java."
— Skynet, probably
<blockquote>
  "The AI became self-aware. Its first act was to uninstall Java."
  <footer>— Skynet, probably</footer>
</blockquote>

# Typography

**Sutairu** provides a variety of typographic utilities for styling text semantically or with utility classes. Below is a reference of supported styles.

| Style             | Example                                                             | HTML Tag  | CSS Class  |
| ----------------- | ------------------------------------------------------------------- | --------- | ---------- |
| **Bold**          | <b>Bold text</b>                                                    | `<b>`     | `.bold`    |
| **Italic**        | <i>Italic text</i>                                                  | `<i>`     | `.italic`  |
| **Abbreviation**  | <abbr title="Abbreviation">Abbr.</abbr>                             | `<abbr>`  | `.td-abbr` |
| **Strikethrough** | <s>Strikethrough</s>                                                | `<s>`     | `.td-lt`   |
| **Underline**     | <u>Underline</u>                                                    | `<u>`     | `.td-ul`   |
| **No decoration** | <a class="no-decoration" href="#typography">Not underlined link</a> | -         | `.td-none` |
| **Small**         | <small>Small</small>                                                | `<small>` | -          |
| **Monospace**     | <span class="ff-mono">Monospace</span>                              | -         | `.ff-mono` |
| **Inline code**   | <code>Inline Code</code>                                            | `<code>`  | -          |
| **Keyboard**      | <kbd>Ctrl</kbd> + <kbd>F</kbd>                                      | `<kbd>`   | -          |
| **Subscript**     | Regular<sub>sub</sub>                                               | `<sub>`   | -          |
| **Superscript**   | Regular<sup>sup</sup>                                               | `<sub>`   | -          |


The table above lists the core utilities that form the foundation of Sutairu’s typography. Additional typography utilities are available in the <a href="#extended-typography" data-close-modal>Extended Typography</a> package.

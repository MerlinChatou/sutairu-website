# Typography

**Sutairu** provides a variety of typographic utilities for styling text semantically or with utility classes. Below is a reference of supported styles.

| Style             | Example                                                             | HTML Tag  | CSS Class         |
| ----------------- | ------------------------------------------------------------------- | --------- | ----------------- |
| **Bold**          | <b>Bold text</b>                                                    | `<b>`     | `.bold`           |
| **Italic**        | <i>Italic text</i>                                                  | `<i>`     | `.italic`         |
| **Abbreviation**  | <abbr title="Abbreviation">Abbr.</abbr>                             | `<abbr>`  | `.td-abbr`        |
| **Strikethrough** | <s>Strikethrough</s>                                                | `<s>`     | `.td-l`           |
| **Underline**     | <u>Underline</u>                                                    | `<u>`     | `.td-u`           |
| **Overline**      | <span class="td-o">Overline</span>                                  | -         | `.td-o`           |
| **No decoration** | <a class="no-decoration" href="#typography">Not underlined link</a> | -         | `.td-none`        |
| **Small**         | <small>Small</small>                                                | `<small>` | -                 |
| **Monospace**     | <span class="ff-mono">Monospace</span>                              | -         | `.ff-mono`        |
| **Handwritten**   | <span class="ff-handwritten">Handwritten</span>                     | -         | `.ff-handwritten` |
| **Highlight**     | <mark>Highlighted text</mark>                                       | `<mark>`  | `.highlight`      |
| **Inline code**   | <code>Inline Code</code>                                            | `<code>`  | -                 |
| **Keyboard**      | <kbd>Ctrl</kbd> + <kbd>F</kbd>                                      | `<kbd>`   | -                 |
| **Subscript**     | Regular<sub>sub</sub>                                               | `<sub>`   | -                 |
| **Superscript**   | Regular<sup>sup</sup>                                               | `<sup>`   | -                 |
| **Deleted**       | <del>Removed text</del>                                             | `<del>`   | `.td-del`         |
| **Inserted**      | <ins>Added text</ins>                                               | `<ins>`   | `.td-ins`         |
| **Uppercase**     | <span class="tt-upper">uppercase text</span>                        |           | `.tt-upper`       |
| **Lowercase**     | <span class="tt-lower">LOWERCASE TEXT</span>                        |           | `.tt-lower`       |
| **Capitalize**    | <span class="tt-capitalize">capitalize each word</span>             |           | `.tt-capitalize`  |
| **Normal case**   | <span class="tt-normal">Normal Case</span>                          |           | `.tt-normal`      |

# Heading Classes

Sutairu provides `.h1` through `.h6` utility classes that allow you to decouple the visual styling of a heading from the semantic structure of your HTML. This ensures you can maintain SEO-friendly tagging while achieving the desired design scale.


## Usage

Apply a `.hx` class to any element (including `<p>`, `<span>`, or `<div>`) to give it the font size, line height, and weight of the corresponding heading level.

<div class="card p-3 w-fit">
<h1 class="h1">Main Heading</h1>
<p class="h2">Big Hero Text</p>
<h2 class="h3">Sub-section Title</h2>
</div>


```html
<h1 class="h1">Main Heading</h1>
<p class="h2">Big Hero Text</p>
<h2 class="h3">Sub-section Title</h2>
```
